## Installation

Add this line to your application's Gemfile:

    gem 'devise'
    gem 'omniauth'
    gem 'omniauth-facebook'
    gem 'oauth_bundle'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install oauth_bundle

## Usage



Setup before start to use it.
rails g devise:install
rails g devise user
rails g oauth_bundle:install
